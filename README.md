# RankPilot：四川高考志愿冲稳保模拟器

RankPilot 是一个用于四川省高考志愿填报参考的交互式模拟工具。用户输入自己的高考分数和可选位次后，可以根据城市、学校标签、学校名称和专业关键词筛选高校专业，并查看系统生成的“冲、稳、保”分类结果。

本项目基于四川省 2025 年物理类录取数据制作，适合用于快速了解不同高校和专业与个人分数之间的距离。项目仅用于课程演示和数据可视化练习，不构成真实志愿填报建议。

## 主要功能

- 输入高考分数，计算个人分数与各专业/专业组录取线之间的分数差。
- 可选输入全省位次，查看个人位次与最低录取位次之间的差距。
- 按城市、学校标签、指定学校和专业关键词筛选候选项。
- 根据保守、均衡、激进三种风险偏好生成“冲、稳、保”分类。
- 展示候选学校和专业的最低分、最低位次、分数差、位次差和数据备注。
- 通过图表直观查看不同候选项与个人分数之间的距离。
- 提供规则版解释建议，并可在配置条件满足时生成 AI 补充解释。

## 使用方法

1. 输入自己的高考分数。
2. 如有需要，输入自己的全省位次；如果不使用位次，可保持为 0。
3. 选择风险偏好：保守、均衡或激进。
4. 根据需要选择城市、学校标签、指定学校或输入专业关键词。
5. 点击“生成冲稳保方案”。
6. 查看结果表格、分数差图表和解释建议。

## 冲稳保规则

系统使用：

```text
分数差 = 你的分数 - 录取线
```

三种风险偏好的规则如下：

| 风险偏好 | 冲 | 稳 | 保 |
| --- | --- | --- | --- |
| 保守 | 分数差 < 0 | 0 <= 分数差 <= 15 | 分数差 > 15 |
| 均衡 | 分数差 < -5 | -5 <= 分数差 <= 10 | 分数差 > 10 |
| 激进 | 分数差 < -10 | -10 <= 分数差 <= 5 | 分数差 > 5 |

如果输入了位次，系统还会计算：

```text
位次差 = 最低位次 - 你的位次
```

位次越小越好，因此该差值越大，说明相对越稳。

## 数据说明

项目数据为四川省 2025 年物理类高校录取线，当前包含 43 所高校、1033 条记录。不同高校公开数据口径可能包括专业线、专业类线、专业组线或院校最低线。

部分记录原表自带最低位次；缺失位次的记录使用四川省 2025 年物理类一分一段表补齐。补齐时只使用物理类部分，读取分数和累计位次，不使用历史类数据。

## 注意事项

- 本项目只基于 2025 年录取数据进行静态模拟，不能代表未来年份的真实录取结果。
- 录取线会受到招生计划、报考热度、专业组设置、选科要求等因素影响。
- 不同高校公开数据口径不完全一致，结果只能作为初步参考。
- 实际填报前，应再次核对四川省教育考试院、高校招生章程和高校官方录取数据。
